## 상품 관리 (productStorage.js)

### 파일: `src/utils/productStorage.js`

#### 데이터 매핑 (DB → Frontend)

```javascript
function toProduct(row) {
  return {
    id:            row.id,
    slug:          row.slug,
    category:      row.category,       // 'single-site' | 'bundle' | 'custom'
    title:         row.title,           // 한국어 제목
    titleEn:       row.title_en,        // 영어 제목
    description:   row.description,
    descriptionEn: row.description_en,
    price:         row.price,           // 가격 (원)
    imageUrl:      row.image_url,
    isSoldOut:     row.is_sold_out,
    isActive:      row.is_active,
    sortOrder:     row.sort_order,
    createdAt:     row.created_at,
    updatedAt:     row.updated_at
  };
}
```

#### API 함수

| 함수 | 설명 | 비고 |
|------|------|------|
| `getProducts(includeInactive)` | 상품 목록 조회 | 기본: 활성만, 관리자: 전체 |
| `getProduct(id)` | 단일 상품 조회 | - |
| `createProduct(data)` | 상품 등록 | 관리자 전용 |
| `updateProduct(id, updates)` | 상품 수정 | 관리자 전용 |
| `deleteProduct(id)` | 상품 삭제 (소프트) | is_active = false |
| `toggleSoldOut(id, isSoldOut)` | 품절 토글 | 관리자 전용 |

## 쇼핑 카트 (CartContext)

### 파일: `src/contexts/CartContext.tsx`

#### localStorage 키
```
dreamitbiz_aihub_cart
```

#### Context API

```typescript
interface CartContextType {
  cartItems: any[];           // 장바구니 상품 목록
  cartTotal: number;          // 총 금액
  cartCount: number;          // 총 수량
  addItem: (product) => void; // 상품 추가 (이미 있으면 수량+1, 최대 99)
  removeItem: (id) => void;   // 상품 삭제
  updateQuantity: (id, qty) => void; // 수량 변경 (1~99)
  clearCart: () => void;      // 장바구니 비우기
}
```

#### 동작 원리
1. 초기화: `localStorage.getItem('dreamitbiz_aihub_cart')` → JSON.parse
2. 상품 추가: 동일 상품이면 수량 +1 (최대 99), 새 상품이면 quantity: 1
3. 품절 상품: `isSoldOut === true`이면 추가 불가
4. 변경 시: `localStorage.setItem()`으로 자동 동기화
