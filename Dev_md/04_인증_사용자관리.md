# AI Hub 인증 및 사용자 관리

## 인증 방식

| 항목 | 내용 |
|------|------|
| 인증 제공자 | Supabase Auth |
| OAuth 지원 | Google, Kakao |
| 세션 관리 | 자동 갱신 (autoRefreshToken: true) |
| Auth Flow | PKCE (Proof Key for Code Exchange) |
| 세션 유지 | persistSession: true (localStorage) |
| 공유 범위 | 11개 학습 사이트 + AI Hub 동일 Supabase 인스턴스 |

## AuthContext 구조

### 파일: `src/contexts/AuthContext.tsx`

```typescript
interface AuthContextType {
  user: any;              // Supabase auth.users 객체
  profile: any;           // user_profiles 레코드
  loading: boolean;       // 초기 세션 로드 중
  isLoggedIn: boolean;    // user 존재 여부
  isAdmin: boolean;       // 관리자 이메일 여부
  signOut: () => Promise<void>;
  refreshProfile: () => Promise<void>;
  accountBlock: { status, reason, suspended_until } | null;
  clearAccountBlock: () => void;
}
```

### 초기화 플로우
```
AuthProvider mount
  ↓
client.auth.onAuthStateChange()
  ├── INITIAL_SESSION → setLoading(false) (5초 타임아웃 fallback)
  ├── SIGNED_IN → loadProfile() + last_sign_in_at 업데이트
  └── SIGNED_OUT → setProfile(null)
```

### 프로필 로드 로직 (`loadProfile`)
1. `getProfile(user.id)` → user_profiles 조회
2. 자동 업데이트 필드:
   - `signup_domain`: 최초 가입 도메인 기록
   - `role`: 'user' → 'member'로 승격
   - `visited_sites`: 현재 도메인 추가 (방문 이력)
3. `check_user_status` RPC 호출 → 차단/정지 상태 확인
   - 차단 시: 강제 로그아웃 + accountBlock 상태 설정

## 관리자 판별

### 파일: `src/config/admin.js`
```javascript
export const ADMIN_EMAILS = ['aebon@kakao.com', 'aebon@kyonggi.ac.kr'];
```

### 관리자 판별 로직 (AuthContext)
```javascript
const allEmails = [
  user?.email,
  user?.user_metadata?.email,
  user?.identities?.[0]?.identity_data?.email,
  profile?.email,
].filter(Boolean).map(e => e.toLowerCase());

const isAdmin = allEmails.some(e => ADMIN_EMAILS.includes(e));
```
- OAuth 로그인 시 이메일이 여러 위치에 분산되므로 모든 소스를 확인

## 사용자 프로필 관리

### 파일: `src/utils/auth.js`

| 함수 | 설명 |
|------|------|
| `getProfile(userId)` | user_profiles에서 프로필 조회 (없으면 자동 생성) |
| `updateProfile(userId, updates)` | display_name, avatar_url 등 수정 |
| `signOut()` | Supabase 로그아웃 |

## 관리자 API

### 파일: `src/utils/adminApi.js`

| 함수 | 설명 |
|------|------|
| `getDashboardStats()` | 대시보드 통계 (주문수, 매출, 회원수, 상품수) |
| `getAllOrders({ page, limit, status })` | 전체 주문 조회 + 필터 |
| `updateOrderStatus(orderId, status)` | 주문 상태 변경 |
| `getAllMembers({ page, limit, search })` | 전체 회원 조회 + 검색 |
| `updateMemberStatus(userId, status, reason)` | 회원 상태 변경 (차단/정지) |
