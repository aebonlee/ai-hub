# AI Hub 관리자 대시보드

## 접근 제어

### 관리자 이메일
```javascript
// src/config/admin.js
export const ADMIN_EMAILS = ['aebon@kakao.com', 'aebon@kyonggi.ac.kr'];
```

### 접근 조건
- OAuth 로그인 후 이메일이 `ADMIN_EMAILS`에 포함되면 `isAdmin = true`
- 관리자 레이아웃(`AdminLayout.tsx`) 및 가드(`AdminGuard`) 적용

---

## 관리자 페이지 목록

| 경로 | 컴포넌트 | 기능 |
|------|----------|------|
| `/admin/dashboard` | Dashboard.tsx | 통계 대시보드 |
| `/admin/orders` | Orders.tsx | 주문 관리 |
| `/admin/members` | Members.tsx | 회원 관리 |
| `/admin/products` | Products.tsx | 상품 관리 |
| `/admin/products/new` | ProductForm.tsx | 상품 등록 |
| `/admin/products/edit/:id` | ProductForm.tsx | 상품 수정 |

## Dashboard (통계)

### API: `getDashboardStats()`

| 지표 | 설명 | 계산 방법 |
|------|------|-----------|
| 주문수 | 전체 주문 건수 | `ah_orders` COUNT |
| 매출 | 결제 완료 매출 합계 | `status='paid'`의 `total_amount` SUM |
| 회원수 | 전체 회원 수 | `user_profiles` COUNT |
| 상품수 | 활성 상품 수 | `ah_products WHERE is_active=true` COUNT |

---

## Orders (주문 관리)

### API: `getAllOrders({ page, limit, status })`

| 기능 | 설명 |
|------|------|
| 필터 | 전체 / 결제완료 / 대기 / 취소 / 환불 |
| 페이지네이션 | 20건/페이지 |
| 상태 변경 | 드롭다운으로 즉시 변경 |
| 상세 보기 | 확장형 주문 상세 (주문 아이템 목록) |

### 주문 상태값
- `pending` — 결제 대기
- `paid` — 결제 완료
- `failed` — 결제 실패
- `cancelled` — 주문 취소
- `refunded` — 환불 완료

## Products (상품 관리)

| 기능 | 설명 |
|------|------|
| 상품 목록 | 전체 상품 (비활성 포함) |
| 상품 등록/수정 | ProductForm 컴포넌트 |
| 품절 토글 | is_sold_out 토글 |
| 삭제 | 소프트 삭제 (is_active = false) |
