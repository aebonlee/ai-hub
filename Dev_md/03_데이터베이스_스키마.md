## 테이블 상세 스키마

### 1. `ah_products` — 상품 (분양 상품)
```sql
id          BIGSERIAL PRIMARY KEY
slug        TEXT UNIQUE                -- 상품 슬러그
category    TEXT DEFAULT 'single-site' -- single-site | bundle | custom
title       TEXT NOT NULL              -- 한국어 제목
title_en    TEXT                       -- 영어 제목
description TEXT                       -- 한국어 설명
description_en TEXT                    -- 영어 설명
price       INTEGER DEFAULT 0         -- 가격 (원)
image_url   TEXT                       -- 상품 이미지 URL
is_sold_out BOOLEAN DEFAULT FALSE      -- 품절 여부
is_active   BOOLEAN DEFAULT TRUE       -- 활성/비활성 (소프트 삭제)
sort_order  INTEGER DEFAULT 0         -- 정렬 순서
created_at  TIMESTAMPTZ DEFAULT NOW()
updated_at  TIMESTAMPTZ DEFAULT NOW()
```

**카테고리 분류:**
- `single-site`: 개별 사이트 분양 (50만원)
- `bundle`: 11개 사이트 번들 (300만원)
- `custom`: 맞춤 제작 (문의)

### 2. `ah_orders` — 주문
```sql
id                  UUID PRIMARY KEY DEFAULT gen_random_uuid()
order_number        TEXT UNIQUE NOT NULL  -- AIH-YYMMDD-XXXXX
user_id             UUID FK → auth.users
user_email          TEXT NOT NULL
user_name           TEXT NOT NULL
user_phone          TEXT
total_amount        INTEGER DEFAULT 0
payment_method      TEXT DEFAULT 'card'   -- card | transfer
payment_status      TEXT DEFAULT 'pending'-- pending|paid|failed|cancelled|refunded
portone_payment_id  TEXT                  -- iamport imp_uid
paid_at             TIMESTAMPTZ
cancelled_at        TIMESTAMPTZ
cancel_reason       TEXT
status              TEXT DEFAULT 'pending'
created_at          TIMESTAMPTZ DEFAULT NOW()
updated_at          TIMESTAMPTZ DEFAULT NOW()
```

### 3. `ah_order_items` — 주문 상세
```sql
id             BIGSERIAL PRIMARY KEY
order_id       UUID FK → ah_orders(id) ON DELETE CASCADE
product_title  TEXT NOT NULL
quantity       INTEGER DEFAULT 1
unit_price     INTEGER DEFAULT 0
subtotal       INTEGER DEFAULT 0
created_at     TIMESTAMPTZ DEFAULT NOW()
```

### 4. `ah_notices` — 공지사항
```sql

title       TEXT NOT NULL
title_en    TEXT

content_en  TEXT
is_pinned   BOOLEAN DEFAULT FALSE

author_name TEXT

created_at  TIMESTAMPTZ DEFAULT NOW()
updated_at  TIMESTAMPTZ DEFAULT NOW()
```

### 5. `ah_qna_posts` — Q&A 질문
```sql

title        TEXT NOT NULL

is_answered  BOOLEAN DEFAULT FALSE

created_at   TIMESTAMPTZ DEFAULT NOW()
updated_at   TIMESTAMPTZ DEFAULT NOW()
```

### 6. `ah_comments` — 댓글
```sql
id          BIGSERIAL PRIMARY KEY
post_id     BIGINT NOT NULL
post_type   TEXT NOT NULL DEFAULT 'qna'
author_id   UUID FK → auth.users
author_name TEXT
content     TEXT NOT NULL
created_at  TIMESTAMPTZ DEFAULT NOW()
```

### 7. `ah_franchise_applications` — 분양 신청

### 8~10. 블로그/게시판/갤러리 — 공통 구조
- 다국어 지원 (title/title_en, content/content_en)
- 작성자 연결 (author_id → auth.users)
- 조회수 카운트, 생성/수정 타임스탬프

### 11. `user_profiles` — 회원 프로필 (공유)


## RLS (Row Level Security) 정책

### 공개 읽기 (Public Read)
- `ah_products`, `ah_notices`, `ah_qna_posts`, `ah_comments`
- `ah_blog_posts`, `ah_board_posts`, `ah_gallery_items`

### 인증된 사용자 쓰기 (Authenticated Insert)
- `ah_orders` — `auth.uid() IS NOT NULL`
- `ah_order_items` — 무조건 허용 (주문과 함께 생성)
- `ah_qna_posts`, `ah_comments` — `auth.uid() = author_id`
- `ah_franchise_applications` — 무조건 허용

### 소유자 수정/삭제 (Owner Update/Delete)
- `ah_orders` — `auth.uid() = user_id` (수정)
- `ah_qna_posts` — `auth.uid() = author_id` (수정/삭제)
- `ah_comments` — `auth.uid() = author_id` (삭제)

### 주문 조회
- `ah_orders` — 자신의 주문만 조회 가능 (`auth.uid() = user_id`)
- `ah_order_items` — 모든 인증 사용자 조회 가능
