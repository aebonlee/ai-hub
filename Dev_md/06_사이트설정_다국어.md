## 학습 사이트 데이터 구조

각 학습 사이트는 다음 필드를 포함합니다:

```javascript
{
  id: 'chatgpt',                    // 고유 식별자
  name: 'ChatGPT 활용',              // 한국어 이름
  nameEn: 'ChatGPT Utilization',     // 영어 이름
  url: 'https://chatgpt.dreamitbiz.com', // 사이트 URL
  icon: 'fa-solid fa-message',       // FontAwesome 아이콘
  color: '#10A37F',                  // 브랜드 컬러
  category: 'chatbot',              // 소속 카테고리
  description: '...',                // 한국어 설명
  descriptionEn: '...',             // 영어 설명
  techStack: ['ChatGPT', 'GPT-4o'], // 기술 스택
  difficulty: 'beginner',           // beginner | intermediate | advanced
  curriculum: [...],                 // 커리큘럼 (한국어)
  curriculumEn: [...],              // 커리큘럼 (영어)
  features: [...],                   // 특징 (한국어)
  featuresEn: [...],                // 특징 (영어)
  target: '...',                     // 대상 학습자 (한국어)
  targetEn: '...'                   // 대상 학습자 (영어)
}
```

## 다국어 (i18n) 지원

### 파일: `src/utils/translations.js`
### Context: `src/contexts/LanguageContext.tsx`

#### 지원 언어
- `ko` — 한국어 (기본)
- `en` — 영어

#### 사용법
```typescript
const { language, t } = useLanguage();
const isEn = language === 'en';

// 번역 키 사용
<h1>{t('site.home.title')}</h1>

// 직접 분기
<span>{isEn ? product.titleEn : product.title}</span>
```

#### 주요 번역 키 네임스페이스
| 접두사 | 용도 |
|--------|------|
| `site.home.*` | 홈페이지 |
| `site.nav.*` | 네비게이션 |
| `shop.*` | 쇼핑/상품 |
| `checkout.*` | 결제 |
| `order.*` | 주문 |
| `auth.*` | 인증/마이페이지 |
| `community.*` | 커뮤니티 |
| `search.*` | 검색 |
